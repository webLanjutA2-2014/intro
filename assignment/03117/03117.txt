--> untuk mengetahui status atau kondisi di dalam repositori yang telah dibuat

3. git add '*.txt' 
--> untuk menambahkan file ke stagging area

--> untuk memberi tahu kepasd git tentang perubahan, untuk menambahkan file ke dalam staging area  shg git bisa melacat perubahan area

5. git commit -m "text" 
--> untuk menyimpan perubahan stage 

6. git log 
--> untuk melihat catatan aktifitas yang telah kita lakukan

7. git remote add origin https

18. git checkout master
